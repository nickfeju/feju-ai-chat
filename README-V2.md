# FéjuAI Web App V2

De webapp blijft hetzelfde `/api/ask` contract gebruiken. Nieuw:

- `bar-list` analyticsweergave wordt ondersteund;
- live tickets kunnen als compacte ticketcards worden getoond;
- kritieke tickets krijgen een duidelijke markering.

Build:

```powershell
npm install
npm run build
```
